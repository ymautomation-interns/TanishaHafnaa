# Employee Scoring System

A small React app with three sections:
1. Scoring Guide — the automatic scoring formula and rating logic reference.
2. Employee Card — profile card that expands into a Performance Rated Criteria panel.
3. Overall Average — a live gauge/dashboard driven by the criteria ratings.

## Run it

```bash
npm install
npm run dev
```

Then open the local URL Vite prints (usually http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```
